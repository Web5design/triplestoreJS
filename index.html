TriplestoreJS

Triplestore wrapper for HTML5 WebStorage

Welcome to TriplestoreJS

This JavaScript wrapper library enables web applications to store triples subject-property-value into HTML5 Web Storage. The API looks like an extension of W3C RDFa API.

  • Functionality
    • Save semantic data as triples into WebStorage
    • Get/Edit/Remove titples stored in WebStorage

Create Triplestore instance

Call simple constructor.

var st = new Triplestore();

Setup prefix mapping

Setup prefix and the corresponding URI mapping for further operations.

st.setMapping("foaf", "http://xmlns.com/foaf/0.1/");

Store triples

push function saves specified triples into WebStorage.

st.push("http://sample.org/people/#danbri", "foaf:name", "Dan Brickley");
st.push("http://sample.org/people/#danbri", "foaf:homepage", "http://danbri.org/");

Retrieve stored triples

get* functions retrieve stored triples in WebStorage.

var subjects = st.getSubjects();
for(var i = 0; i < subjects.length; i++) {
  var subject = subjects[i];
  var properties = st.getProperties(subject);
  for(var j = 0; j < properties.length; j++) {
    var property = properties[j];
    var value = st.getValues(subject, property);
    console.log(subject, property, value);
  }
}

Remove stored triples

remove function delete the matched triples from WebStorage.

Remove a subject

this.st.remove("http://sample.org/people/#danbri");

Remove a property of a specified subject

this.st.remove("http://sample.org/people/#danbri", "foaf:homepage");

Remove a property from each subject

this.st.remove(null, "foaf:homepage");

Remove all subjects

this.st.remove();
